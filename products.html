PDCloudEX SDN
PDCloudEX SDN
PDCloudEx SDN is a software-defined networking solution which provides an intra-datacenter connectivity for virtual and physical workloads on the cloud. Built on OpenDayLight, OpenVSwitch and Openstack, the product provides a disaggregation for both software and hardware components in a network, while providing full control over the virtual networks. Our technology enables full network virtualization and allows enterprises, data centers and service providers to easily deploy, control, monitor and manage secure multi-tenant network infrastructure. In other words, it is a comprehensive solution that makes the network as readily consumable as compute resources across the datacenter, enterprise WAN and public cloud providers.
